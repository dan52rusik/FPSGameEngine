# μfps

Маленькая FPS игра, написанная на современном C++ и OpenGL с нуля, в прямом эфире.

Присоединяйтесь к нам на [Discord](https://discord.gg/9FkkMgXSUV) для обсуждения проекта.

## "С нуля"

Где возможно мы будем строить всё с нуля. Это включает: математику, оконную систему, рендеринг и т.д.

Небольшие уступки, которые я готов сделать:
- Парсинг форматов файлов (например, загрузка изображений, моделей)
- Физика (слишком сложная для такого маленького проекта)
- Lua скриптинг (хотя мы будем писать свои собственные биндинги)

## Сборка

Попробуйте собрать проект самостоятельно, но я не буду распространять какие-либо ассеты, на которые у меня нет лицензии.

Система сборки использует Docker, и вы можете использовать Makefile верхнего уровня

```bash
# собрать docker образ (однократно)
make docker-image

# настроить проект
make configure

# собрать и запустить игру
make run

# (опционально) собрать без запуска
make build

# запустить тесты
make tests
```
